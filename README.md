# StockBuster

Movie search app using OMDB API.

## Features

- Search movies with autocomplete
- Infinite scroll (custom, no plugins)
- Movie detail page
- Poster lightbox modal

## Stack

React 19 (Next.js), Redux Toolkit, Axios, Tailwind CSS, Jest

## Run Locally

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000

## Tests

```bash
pnpm test
```
